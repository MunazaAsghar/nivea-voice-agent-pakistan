# nivea-voice-agent-pakistan
Bilingual (Urdu/English) AI voice agent using Retell AI and Twilio SIP for automated order processing and customer support in NIVEA Pakistan.
Voice Agent in Retell AI (NIVEA Pakistan)
📌 Overview

This project implements a bilingual (Urdu/English) AI voice agent built using Retell AI and Twilio Elastic SIP Trunking. It is designed for NIVEA Pakistan to automate customer interactions including order placement and complaint handling.
